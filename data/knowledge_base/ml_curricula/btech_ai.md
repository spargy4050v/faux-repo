# BTech in Artificial Intelligence - 8 Semester Curriculum

**Level:** BTech  
**Duration:** 4 Years (8 Semesters)  
**Total Credits:** 160

## Overview
A comprehensive undergraduate program combining computer science fundamentals with specialized AI and machine learning courses. Prepares students for careers in AI development, research, and applications.

## Semester 1 (20 Credits)
- **CS101** - Programming Fundamentals (4 credits) - Core
- **MA101** - Calculus I (4 credits) - Core
- **PH101** - Physics I (3 credits) - Core
- **EN101** - English Communication (3 credits) - Core
- **CS102L** - Programming Lab (3 credits) - Lab
- **PH102L** - Physics Lab (2 credits) - Lab
- **PE101** - Physical Education (1 credit) - Core

## Semester 2 (20 Credits)
- **CS201** - Data Structures (4 credits) - Core - Prerequisites: CS101
- **MA201** - Linear Algebra (4 credits) - Core
- **MA202** - Probability and Statistics (3 credits) - Core
- **CS202** - Digital Logic Design (3 credits) - Core
- **CS203L** - Data Structures Lab (3 credits) - Lab - Prerequisites: CS101
- **HS201** - Professional Ethics (2 credits) - Core
- **PE201** - Physical Education (1 credit) - Core

## Semester 3 (20 Credits)
- **CS301** - Algorithms (4 credits) - Core - Prerequisites: CS201
- **CS302** - Database Systems (3 credits) - Core
- **AI301** - Introduction to AI (4 credits) - Core - Prerequisites: CS201
- **MA301** - Discrete Mathematics (3 credits) - Core
- **CS303L** - Algorithm Lab (3 credits) - Lab - Prerequisites: CS201
- **AI302L** - AI Programming Lab (3 credits) - Lab

## Semester 4 (20 Credits)
- **AI401** - Machine Learning (4 credits) - Core - Prerequisites: AI301, MA202
- **CS401** - Operating Systems (3 credits) - Core
- **CS402** - Computer Networks (3 credits) - Core
- **AI402** - Neural Networks (3 credits) - Core - Prerequisites: AI301
- **AI403L** - ML Lab (4 credits) - Lab - Prerequisites: AI301
- **HS401** - Technical Writing (3 credits) - Core

## Semester 5 (20 Credits)
- **AI501** - Deep Learning (4 credits) - Core - Prerequisites: AI401, AI402
- **AI502** - Natural Language Processing (3 credits) - Core - Prerequisites: AI401
- **AI503** - Computer Vision (3 credits) - Core - Prerequisites: AI401
- **CS501** - Software Engineering (3 credits) - Core
- **AI504L** - Deep Learning Lab (4 credits) - Lab - Prerequisites: AI401
- **AI505** - Elective I (3 credits) - Elective

## Semester 6 (20 Credits)
- **AI601** - Reinforcement Learning (3 credits) - Core - Prerequisites: AI401
- **AI602** - Knowledge Representation (3 credits) - Core
- **AI603** - AI Ethics and Safety (3 credits) - Core
- **CS601** - Cloud Computing (3 credits) - Core
- **AI604** - Elective II (3 credits) - Elective
- **AI605** - Mini Project (5 credits) - Lab - Prerequisites: AI501

## Semester 7 (20 Credits)
- **AI701** - Advanced ML Topics (3 credits) - Core - Prerequisites: AI501
- **AI702** - MLOps (3 credits) - Core - Prerequisites: AI501
- **AI703** - Elective III (3 credits) - Elective
- **AI704** - Elective IV (3 credits) - Elective
- **AI705** - Industry Internship I (4 credits) - Lab
- **AI706** - Industry Internship II (4 credits) - Lab

## Semester 8 (20 Credits)
- **AI801** - Capstone Project I (4 credits) - Lab - Prerequisites: 140 credits completed
- **AI802** - Capstone Project II (4 credits) - Lab - Prerequisites: 140 credits completed

- **AI805** - Elective V (3 credits) - Elective
- **AI806** - Emerging Trends in AI (4 credits) - Core
